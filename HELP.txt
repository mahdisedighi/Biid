روی یک سیستم جدید:

1- روی آخرین سیستم push بزنیم

2- نصب پایتون و جنگو و گیت

3- توی cmd  مسیر فلدر مورد نظر این کد :

git clone https://github.com/mahdisedighi/Biid.git
(فایل های کد و دیتابیس را دانلود میکند)

4- ران کردن origin  - (پیش نیاز گیت : برای شناسایی لینک گیت هاب)

5- در cmd این کد : 
git config --global user.email "email"
git config --global user.name "username"
